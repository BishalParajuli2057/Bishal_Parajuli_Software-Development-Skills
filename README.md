# Grocery Manager App

Bishal Parajuli

## Description

Grocery Manager App is an Android application developed using Java and Android Studio. The application allows users to manage grocery items through a simple and user-friendly interface.

### Features

* Add grocery items
* Add notes for grocery items
* Mark grocery items as important
* View all grocery items in a RecyclerView
* Navigate between pages using tabs
* Automatically clear input fields after adding an item

## Technologies Used

* Java
* Android Studio
* RecyclerView
* Fragments
* ViewPager2
* Material Design Components

## Project Structure

The application consists of three main views:

1. Main Page
2. Add Grocery Page
3. Grocery List Page

## How to Run the Project

### Requirements

* Android Studio installed on your computer
* Android Emulator or Android Device

### Steps

1. Download the project ZIP file.
2. Extract (unzip) the project to a folder on your computer.
3. Open Android Studio.
4. Select Open from the Android Studio welcome screen.
5. Browse to the extracted project folder and open it.
6. Wait for Gradle synchronisation to complete.
7. Connect an Android device or start an Android Emulator.
8. Click the Run button in Android Studio.
9. The application will build and launch automatically.

## Application Usage

* Open the Add Grocery tab.
* Enter a grocery name and an optional note.
* Mark the item as important if needed.
* Press **Add Grocery**.
* Open the **All Items** tab to view the grocery list.
